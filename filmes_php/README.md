# CRUD Filmes - Gêneros e Filmes
Projeto tema 'Filmes' para a disciplina Linguagem e Técnicas de Programação — Professor José Carlos Domingues Flores.

## Como usar
1. Copie a pasta `filmes_php` para `htdocs` do XAMPP (ex: C:/xampp/htdocs/).
2. Importe `database.sql` no phpMyAdmin para criar o banco `filmes_db` e dados de exemplo.
3. Inicie Apache e MySQL no XAMPP.
4. Acesse: http://localhost/filmes_php/public/
