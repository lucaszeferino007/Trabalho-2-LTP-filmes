# CRUD Filmes - Gêneros e Filmes
Projeto tema 'Filmes' para a disciplina Linguagem e Técnicas de Programação — Professor José Carlos Domingues Flores.
Repositório contendo o projeto de banco de dados de filmes, desenvolvido para fins acadêmicos/profissionais. O sistema organiza informações sobre filmes, incluindo títulos, gêneros, diretores, elenco e avaliações, permitindo consultas e análises estruturadas.
Funcionalidades:
Armazenamento de filmes, diretores, atores e gêneros.
Consultas SQL para filtrar, ordenar e agrupar dados.
Estrutura de banco de dados relacional eficiente e normalizada.
Tecnologias utilizadas:
SQL (MySQL/PostgreSQL/SQLite)
Scripts de criação de tabelas e inserção de dados
Consultas e relatórios exemplificativos
Projeto ideal para estudo de modelagem de dados, consultas SQL e gerenciamento de bancos de dados relacionais.

## Como usar
1. Copie a pasta `filmes_php` para `htdocs` do XAMPP (ex: C:/xampp/htdocs/).
2. Importe `database.sql` no phpMyAdmin para criar o banco `filmes_db` e dados de exemplo.
3. Inicie Apache e MySQL no XAMPP.
4. Acesse: http://localhost/filmes_php/public/
